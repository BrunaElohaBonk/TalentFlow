Método de desenvolvimento de rotas:

Autenticação
Método	Rota	Descrição
POST	/login	Login de aprendiz ou instrutor
POST	/logout	Encerrar sessão
POST	/esqueci-senha	Solicitar recuperação de senha
PUT	/alterar-senha	Alterar senha

Instrutores
Método	Rota	Descrição
POST	/instrutores	Cadastrar instrutor
PUT	/instrutores/:edv	Atualizar dados
DELETE	/instrutores/:edv	Excluir instrutor

Turmas
Método	Rota	Descrição
GET	/turmas	Listar turmas
GET	/turmas/:id	Buscar turma
POST	/turmas	Criar turma
PUT	/turmas/:id	Atualizar turma
DELETE	/turmas/:id	Excluir turma


Aprendizes
Método	Rota	Descrição
GET	/aprendizes	Listar aprendizes
GET	/aprendizes/:edv	Buscar aprendiz
POST	/aprendizes	Cadastrar aprendiz
PUT	/aprendizes/:edv	Atualizar aprendiz
DELETE	/aprendizes/:edv	Excluir aprendiz

GET e PUT para cada informação que compõe o perfil
